How To

Run the Web App on your Local Device

Clone the repo to your local device.

Run the command "git clone github-repo-url" in your terminal.

Navigate into the folder containing the code for the Web App.

Run the command "cd LetsMeet"

Install the required node modules.

Run the command "npm install"

Run the Web App.

Run the command "npm start". The crux of the app should be running on localhost:3000.

Start the actual Web App.

Navigate into the public/landing-page file and from there, open the landing.html (Although this works on all major browsers, Chrome Browser recommended for the best experience).

Technologies Used

Node.js

To make a remote connection between two or more devices we need a server. In this case, we need a server that handles real-time communication. we know that Node.js is built for real-time scalable applications. To develop a two-way connection app with free data exchange, we would use WebSockets that allows opening a communication session between a client and a server. Web Real-Time Communications ‚Äì WebRTC in short ‚Äì is an HTML5 specification that allows you to communicate in real-time directly between browsers without any third-party plugins. WebRTC can be used for multiple tasks (even file sharing) but real-time peer-to-peer audio and video communication is the primary feature that has been used in the web application using WebRTC.

Socket.IO is a JavaScript library for realtime web applications. It enables realtime, bi-directional communication between web clients and servers
